# Air Reservation System — Projekt rezerwacji miejsc w samolocie

Projekt edukacyjny przedstawiający **system rezerwacji miejsc w samolocie**.  
Celem było stworzenie aplikacji (konsolowej / GUI / webowej – zależnie od implementacji w repo) umożliwiającej:

- wyszukiwanie dostępnych lotów,
- rezerwację i anulowanie miejsc,
- zarządzanie rezerwacjami pasażerów.

Projekt zrealizowany w ramach nauki programowania i zarządzania danymi oraz logiki aplikacyjnej.

---

## 🎯 Cel projektu

Projekt ma na celu przedstawienie umiejętności:

- modelowania danych (loty, pasażerowie, rezerwacje),
- implementacji logiki rezerwacji,
- pracy z bazą danych (np. SQLite / plikową / in-memory),
- obsługi wejścia użytkownika i prezentacji wyników,
- tworzenia prostego systemu CRUD (Create, Read, Update, Delete).
